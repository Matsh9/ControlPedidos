# Control Pedidos

## Descrição do projeto

O **Control Pedidos** é um aplicativo Android desenvolvido em Kotlin utilizando Jetpack Compose, com o objetivo de gerenciar clientes, produtos e pedidos de forma simples e organizada.

O sistema permite o cadastro de usuários, login, controle de pedidos e visualização de dados em tempo real, utilizando arquitetura MVVM e persistência local com Room e DataStore.

---

## ⚙Funcionalidades

- Login e registro de usuários
- Cadastro e gerenciamento de clientes
- Cadastro e gerenciamento de produtos
- Criação e gerenciamento de pedidos
- Tela inicial com resumo do sistema
- Suporte a tema claro/escuro (DataStore)
- Persistência de dados local (Room Database)
- Navegação entre telas (Navigation Compose)

---

## Tecnologias utilizadas

- Kotlin
- Jetpack Compose
- Android Architecture Components (ViewModel, Lifecycle)
- Room Database
- DataStore Preferences
- Navigation Compose
- Coroutines / Flow
